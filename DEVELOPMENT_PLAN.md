# 品牌素材代下平台 - 開發計劃

## 項目概述
為用戶建立一個自動化素材代下平台，對接小靈素材網站，實現品牌化前台、卡密驗證和自動化下載功能。

## 已完成
- ✅ 前台品牌化網站（綠色主題，去除原平台品牌信息）
- ✅ 素材連結輸入和卡密驗證介面
- ✅ 下載結果顯示（成功、失敗、處理中）
- ✅ 響應式設計

## 待開發 - 後端系統

### 1. 卡密驗證模組
**功能：**
- 驗證小靈素材卡密的有效性
- 檢查卡密的剩餘額度和有效期
- 記錄卡密使用歷史

**技術實現：**
- 使用 Playwright 自動化工具登入小靈素材
- 驗證卡密後獲取帳戶信息
- 存儲卡密狀態到本地數據庫（SQLite）

### 2. 自動化下載模組
**功能：**
- 使用提供的卡密登入小靈素材
- 提交客戶的素材連結
- 解析頁面獲取百度網盤下載連結
- 返回下載連結給前台

**技術實現：**
- Playwright 模擬瀏覽器操作
- 處理動態頁面加載
- 異常處理和重試機制

### 3. 後台管理介面
**功能：**
- 配置小靈素材帳號密碼
- 查看卡密使用情況
- 查看下載歷史記錄
- 系統狀態監控

### 4. API 端點
**需要的 API：**
- `POST /api/verify-card` - 驗證卡密
- `POST /api/download` - 提交下載請求
- `GET /api/card-status` - 查詢卡密狀態
- `GET /api/download-history` - 查看下載歷史
- `POST /api/admin/config` - 配置帳號密碼
- `GET /api/admin/status` - 系統狀態
